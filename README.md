## Installation

### Download

Download the latest version from the [Releases](https://github.com/fhluo/translate-gemma-desktop/releases) page.

### Model Setup

Before using the app, please install [Ollama](https://ollama.com/) and pull the TranslateGemma model:

```shell
ollama pull translategemma
```

You can also choose other model variants:

- `translategemma:12b`
- `translategemma:27b`
